# Password Generator (C++)

## Overview
This project is a simple **Password Generator** written in C++.  
It generates random, secure passwords based on user-defined criteria such as length and character types (letters, numbers, symbols).  

The program is designed for users who want to quickly create strong passwords for accounts, applications, or online services.  

---

## Features
- Generate passwords of any desired length.
- Include:
  - Uppercase letters (A-Z)
  - Lowercase letters (a-z)
  - Numbers (0-9)
  - Special characters (e.g., !@#$%^&*)
- Easy-to-use console interface.

---

## How to Use
1. Open the project folder in your C++ IDE (e.g., Code::Blocks, Visual Studio, or any compiler that supports C++).
2. Compile the `password_generator.cpp` file.
3. Run the executable.
4. Enter the desired password length and options when prompted.
5. Copy and use the generated password securely.
