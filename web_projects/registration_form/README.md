# Registration Form Project

## Overview
This project is a simple **Registration Form** created using HTML and CSS.  
It allows users to enter personal information, select account type, upload a profile picture, and submit their details. The form is designed to be responsive and visually appealing with a dark-themed interface.

---

## Features
- Collect user information:
  - First Name
  - Last Name
  - Email
  - Password
- Select account type (Personal or Business)
- Upload a profile picture
- Input age and select how they heard about the site
- Provide a short bio
- Accept terms and conditions before submission
- Styled with CSS for a modern, user-friendly interface

---

## How to Use
1. Open the folder in your preferred code editor (e.g., VS Code, Sublime Text).  
2. Open `index.html` (or the main HTML file) in a web browser.  
3. Fill in the form fields as required.  
4. Click **Submit** to send data (the demo action points to [FreeCodeCamp registration demo](https://register-demo.freecodecamp.org)).

---

## Files
- `index.html` — Main HTML structure of the registration form  
- `registration form.css` — Stylesheet for the form design and layout  

---

## Requirements
- Modern web browser (Chrome, Firefox, Edge, Safari)
- Basic understanding of HTML and CSS to customize

---

## Notes
- The form includes basic validation using HTML attributes (e.g., `required`, `pattern`, `min`, `max`).  
- The design uses a responsive width so the form adapts to different screen sizes.  
- This project is suitable for learning form creation, CSS styling, and basic web design principles.
